Если у вас есть приватный сервер и вы хотите использовать функции на базе API, например команды Discord-бота, сервисы для бизнеса или любые другие автоматизации, вам не нужно писать код.

**Несколько сторонних приложений уже предлагают готовые интеграции на основе API и позволяют участникам сообщества использовать их напрямую.**

<Warning>
  Эти приложения **не создаются и не поддерживаются командой ER:LC Россия и другими сообществами проекта**. Это независимые сторонние продукты, использующие API. Перед использованием обязательно проведите собственную проверку.
</Warning>

## Избранные Приложения Партнёров

* **YaMRP Такси** (https://mrptaxi.site/)
  * Предлагает: сервис такси на сервере Moscow RolePlay и удобные системы управления им.

<CardGroup cols={2} />

<Note>
  **Мы не можем помочь вам выбрать, какое приложение использовать, и не можем предоставлять поддержку для перечисленных выше приложений. Обращайтесь в каналы поддержки каждого приложения за помощью.**
</Note>

Прежде чем использовать любое клиентское приложение:

* Ознакомьтесь с политикой конфиденциальности и условиями использования приложения.
* Убедитесь, что приложение активно поддерживается и имеет надёжное сообщество.

## Вы разработчик?

Если вы хотите создать собственную интеграцию вместо использования готового клиента, перейдите в раздел [Документацию API](/ru/api-reference/), чтобы увидеть все доступные эндпоинты.
