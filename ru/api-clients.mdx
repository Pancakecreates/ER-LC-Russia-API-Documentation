# Клиенты партнёров

> Владельцы серверов, не владеющие программированием, могут управлять своим приватным сервером ER:LC с помощью этих проверенных сторонних приложений, интегрирующих ERLC API.

Если вы владеете приватным сервером и хотите использовать функции на базе API, такие как команды Discord-бота, панели модерации или автоматические оповещения, вам не нужно писать код.

**Несколько известных сторонних приложений создали полнофункциональные интеграции на основе PRC API и делают их доступными для владельцев серверов напрямую.**

<Warning>
  Эти приложения **не создаются и не поддерживаются командой ER:LC**. Это независимые сторонние продукты, интегрировавшие PRC API. Всегда проводите собственное исследование перед использованием платформы.
</Warning>

## Безопасность: защитите свой ключ сервера

Ваш ключ сервера предоставляет полный доступ к API вашего приватного сервера. Тот, кто им владеет, может выполнять команды и читать все данные сервера.

<Danger>
  Делитесь своим `server-key` только с разработчиками или приложениями, которым вы доверяете. Если вы перестали использовать бота или подозреваете, что ваш ключ был раскрыт, немедленно сбросьте его во внутриигровых настройках ER:LC в разделе **ER:LC API**.
</Danger>

При настройке любого клиента партнёра у вас попросят предоставить ключ сервера. Прежде чем это сделать:

* Ознакомьтесь с политикой конфиденциальности и условиями использования приложения
* Убедитесь, что приложение активно поддерживается и имеет надёжное сообщество
