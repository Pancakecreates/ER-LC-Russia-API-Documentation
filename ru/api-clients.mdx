Если вы хотите использовать функции нашего API для автоматизации, настройки Discord-ботов или управления игровым бизнесом, вам совершенно не обязательно уметь программировать.

**Наши партнёры уже предлагают готовые интеграции на основе API, позволяя участникам сообщества использовать их напрямую.**

<Warning>
  Эти сервисы **не создаются и не поддерживаются командой ER:LC Россия и другими сообществами проекта**. Это независимые сторонние продукты, использующие API. Перед использованием обязательно проведите собственную проверку.
</Warning>

## Проверенные сервисы партнёров

* **YaMRP Такси** (https://mrptaxi.site/)
  * Предлагает: сервис такси на сервере Moscow RolePlay и удобные системы управления им.

<CardGroup cols={2} />

<Note>
  **Мы не можем помочь вам с выбором конкретного сервиса и не предоставляем техническую поддержку по перечисленным выше продуктам. По всем вопросам обращайтесь в официальные каналы поддержки каждого сервиса.**
</Note>

Прежде чем использовать любой сторонний сервис:

* Ознакомьтесь с политикой конфиденциальности и условиями использования данного сервиса.
* Убедитесь, что продукт активно поддерживается и имеет надёжное сообщество.

## Вы разработчик?

Если вы хотите создать собственную интеграцию вместо использования готовых решений от партнёров, перейдите в раздел [Документация API](/ru/api-reference/), чтобы изучить все доступные эндпоинты.
