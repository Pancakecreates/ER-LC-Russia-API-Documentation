**API ER:LC Россия** — это мощный интерфейс для разработчиков, который открывает доступ к функциям нашего сообщества и позволяет в реальном времени взаимодействовать с технической системой проекта, включая управление игровым сервером, экономикой и другими возможностями.


## Кто может использовать?

Любой разработчик, работающий с API ER:LC Россия, может получить доступ к нему после рассмотрения тикета. Ключ для аутентификации запросов выдаётся после одобрения обращения.

  **Если вы не разработчик и не работаете с API, эта документация вам не нужна. Вместо неё используйте одно из готовых приложений партнёров, перечисленных ниже.**
</Note>
