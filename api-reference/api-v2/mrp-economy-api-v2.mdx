## Аутентификация API

Все запросы к API должны содержать валидный заголовок **Authentication c MRPAPIKey**.

**Не знаете, где взять API ключ?**\
Инструкция по получению API-ключа доступна здесь: [Документация API V2](/home/for-developers/documentation-api-v2.md).
