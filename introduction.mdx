Узнайте, что такое API приватного сервера ER:LC, что он позволяет создавать и как получить ключ сервера для аутентификации запросов.

API приватного сервера предоставляет программный интерфейс для внешнего мира вашему приватному серверу ER:LC. Как только на вашем сервере появится пакет **ERLC API**, вы сможете запрашивать данные сервера в реальном времени, выполнять внутриигровые команды, получать вебхуки событий в реальном времени и создавать собственные инструменты и интеграции.

## Кто может использовать

Любой владелец приватного сервера, приобретший серверный пакет ERLC API, может использовать этот API. Вам не требуется никакого специального одобрения, кроме владения пакетом. Ключ вашего сервера — учётные данные для аутентификации запросов — доступен напрямую из внутриигровых настроек ER:LC после активации пакета.
